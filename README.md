# KMP Algorithm Visualizer

A simple interactive visualizer for the **Knuth–Morris–Pratt (KMP)** string matching algorithm, built using **Next.js 16**, **TailwindCSS**, and **Framer Motion**.

## Features
- Animated LPS (prefix table) construction  
- Visual step-by-step matching  
- Auto-play, pause, and manual step control  
- User input for custom text & pattern  

## Run Locally
```bash
npm install
npm run dev
```

Open http://localhost:3000
to view the visualizer.
